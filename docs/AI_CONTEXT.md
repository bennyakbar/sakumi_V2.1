# SAKUMI SYSTEM CONTEXT

SAKUMI (Sistem Akuntansi Keuangan Sekolah) adalah sistem
manajemen keuangan sekolah berbasis web.

Aplikasi ini digunakan oleh:

- Admin Tata Usaha (TU)
- Bendahara
- Kepala Sekolah
- Yayasan
- Auditor

SAKUMI mendukung unit sekolah:

- MI
- RA
- DTA

Core Modules:

1. Student Management
2. Invoice Management
3. Payment Recording
4. Settlement Processing
5. Receipt Printing
6. Arrears Monitoring
7. Financial Reporting
